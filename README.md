# Ritik Sharma Portfolio Website

A modern, responsive, animated portfolio website built with plain HTML, CSS, and JavaScript. Ready to deploy on GitHub Pages.

## Live Demo
**Site URL**: https://hritivizc.github.io/ritik-portfolio/

## Features
- Dark Purple/Teal neon theme
- AOS scroll animations
- Interactive particle canvas background
- Animated stat counters
- Responsive design (mobile, tablet, desktop)
- Experience timeline
- Skills showcase
- Contact form with validation

## Tech Stack
- HTML5 / CSS3 / JavaScript (ES6+)
- [AOS 2.3.4](https://michalsnik.github.io/aos/) - scroll animations
- [Font Awesome 6.5](https://fontawesome.com/) - icons
- [Google Fonts](https://fonts.google.com/) - Inter & Syne
